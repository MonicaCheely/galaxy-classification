# [Galaxy Classification](https://github.com/monicacheely/galaxy-classification)

![Galaxy Classification](assets/galaxy_image.jpg)

## Project Overview
The **Galaxy Classification** project is part of my data science journey, exploring Space Zooniverse data. The project focuses on data cleaning, exploratory analysis, visualization, and building basic machine learning models to classify galaxy types.

## Skills & Tools
- Python, Pandas, NumPy
- Data visualization with Matplotlib & Seaborn
- Machine learning with Scikit-learn
- Jupyter notebooks, Git/GitHub

## Future Work
- Expand dataset with additional galaxy images
- Improve model accuracy and experiment with advanced ML algorithms
- Create an interactive dashboard to visualize classification results
